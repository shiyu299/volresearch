# Simple IV Inspector

This is a stripped-down Streamlit app for colleagues:
- loads files from `data/derived`
- builds the ATM aggregate IV index
- shows main chart, single-contract chart, debug table, and drilldown
- removes all factor-related UI and dependencies from the app layer

Usage:
1. Run `setup_simple_env.bat` once if the `st` environment is missing or outdated.
2. Run `run_simple_app.bat`.
