## 4. 结论（论文可直接引用的实证要点）

1. TA605链在当前样本中，大额traded_vega事件可通过`Q99(abs_traded_vega)`稳定定义，事件率约`0.244%`。
2. 事件前置信号以**流动性与微观结构类特征**最稳：`opt_spread_60s`、`no_trade_move_60`、`fut_imb`。
3. 与直觉不同，`iv_f_div_60`在该样本并未提供正向增益，提示其品种/时段依赖性较强。
4. 日内分布显示午盘风险更集中，夜盘在当前窗口贡献很低。
5. 建议实盘或研究中采用“分层阈值+多信号并联”而非单一固定阈值。

## 5. 可执行监控建议（基于本次结果）

- 一级预警：`opt_spread_60s >= Q80` 且 `no_trade_move_60 >= Q80`
- 二级预警：一级成立 + `fut_imb >= Q80`
- 事件确认：`abs_traded_vega >= Q99`
- 极端事件：`abs_traded_vega >= Q99.9`（建议单独统计）

> 注：阈值应按日滚动重算，避免跨日分布漂移。
