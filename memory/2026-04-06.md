- Main conclusion: `v1.5` looks more jittery mostly because it uses only contracts with a fresh quote in that exact second, while the timeseries app default uses `state_adjust/ffill` to keep stale contracts contributing. The app also defaults to `kline_rule=1min`, which visually smooths the series further.

  - 99% abs diff quantile: `v1.5 = 0.03475`, app default `= 0.00143`.
